# Calculator App

This is a simple calculator application implemented in Python using the Tkinter library.

## Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, division.
- Special functions: clear the current expression, square the current expression, take the square root of the current expression.
- Evaluate complex expressions by combining multiple operations.
- Responsive GUI with digit buttons, decimal point, and operator buttons.

## Getting Started

### Prerequisites

- Python installed on your machine.
